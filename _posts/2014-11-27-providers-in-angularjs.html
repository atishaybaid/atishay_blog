Each web application which is build in angularjs consists of  several components such as Controllers,Services Factories ,Modules etc.Now these components interacts together to perform the desired user action.
Here we are going to discuss the provider recipe.
The provider recipe is useful when we have get some information(data) from an external service(such as external api). which requires configuration depending upon certain parameter.
The provider recipe implements a $get method,which gets executed as soon as provider is triggered.
