I found  Call back functions in javascript as the most confusing topic.To understanding the working of call back function correctly, i searched a ton on google.So, here is the summary of my understanding of the call back function..
To understand call back functions we first have to understand that we first have to understand that functions in javascript are objects,hence these functions can be passed as a argument to another function.
Ideally when we make a function call,the corresponding called function gets called and starts it execution,so uo regular flow of code has to wait till the called function gets executed.
Now,JavaScript gives us a option to call the function asynchronously, This the whole idea behind the execution of call back functions. We dont have to wait for the called function to complete its execution,we use call backs to do it asynchronously.
